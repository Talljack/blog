# 欢迎来到我的博客！

这是我博客的第一篇文章。我很高兴能够与你分享我的想法、经验和见解。

## 关于这个博客

这个博客是使用现代化的技术栈构建的：

- **Next.js 15** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **Markdown/MDX** - 内容格式

## 我会分享什么？

在这个博客中，我计划分享：

1. **技术教程** - 编程技巧和最佳实践
2. **项目经验** - 实际开发中的心得体会
3. **行业观察** - 对技术趋势的思考
4. **个人成长** - 学习和职业发展的感悟

## 代码示例

这里是一个简单的 React 组件示例：

```tsx
import React from 'react'

interface WelcomeProps {
  name: string
}

const Welcome: React.FC<WelcomeProps> = ({ name }) => {
  return (
    <div className='p-4 bg-blue-100 rounded-lg'>
      <h1 className='text-2xl font-bold'>Hello, {name}! 👋</h1>
      <p className='mt-2 text-gray-600'>欢迎来到我的博客！</p>
    </div>
  )
}

export default Welcome
```
